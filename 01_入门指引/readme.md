## 🧠 核心概念

- **人工智能 (AI)**：使机器具有人类智能的技术
- **机器学习 (ML)**：从数据中学习的人工智能分支
- **深度学习 (DL)**：使用多层神经网络的学习方法
- **大语言模型 (LLM)**：基于大规模预训练的生成式 AI 模型

## 📚 学习目标

完成本章节学习后，你将：

1. ✅ 理解 AI、ML、DL、LLM 的关系和区别
2. ✅ 掌握 LLM 核心术语（Token、Embedding、Transformer 等）
3. ✅ 了解大模型的发展历程和重要里程碑
4. ✅ 认识当前主流的大模型产品
